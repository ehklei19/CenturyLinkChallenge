There are two REST APIs at this address:

To retrieve current data use an HTML GET with this URL (if running Glassfish 4.1.1):
http://localhost:8080/CenturyLink-war/webresources/servertrack/{server name}

To log a data point use an HTML POST with this URL:
http://localhost:8080/CenturyLink-war/webresources/servertrack/{server name}
and use this JSON fragment to supply logging info:
{ "servername" : "server", "cpuload" : "100.0", "ramload" : "100.0" }
